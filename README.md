> TSLint config for Lambdas style 


<br/>

### Getting Started

1. Install: `yarn add tslint-config-lambdas -D`.

2. Usage in your `tslint.json`:

    ```json
    { "extends": "lambdas" }
    ```

    **OR**, if you are using the `tslint.yml`:
    ```yml
    extends:
      - 'lambdas'
    ```
